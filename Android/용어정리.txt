안드로이드 스튜디오:
그레들: 안드로이드 빌드 자동화를 위한 스크립트
ㄴ그루비: 그레들 스크립트의 일종(예전부터 사용함) 
ㄴ코틀린: 구글에서 사용하는 자바를 대처하는 언어

의존성: 무엇인가 없어서 나는 오류
버퍼언더플로우:
ㄴ버퍼: 메모리덩어리(파일,변수,RAM에 할당된 크기)
ㄴ언더플로우: 파일 사용영역이 10~100인데  10보다 작은 범위에 접근

SDK(소프트웨어디벨롭먼트키트)
API(어플리케이션소프트웨어인터페이스): 함수
플렛폼: 발판? -> 기반 -> OS, 가상머신, 서비스
Android API: 안드로이드용 클래스와 함수의 집합
ㄴ안드로이드 버전마다 다름
API Level: API버전. OS마다 번호가 붙는다.

SDK Tools
adb(안드로이드 디버그 브릿지): 콘솔명령어로. 디바이스를 원격으로 연결하거나 제어함.

가상머신: OS위에서 작동하는 플렛폼.
JVM(자바가상머신)
